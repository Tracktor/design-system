# `ThemeProvider`

Provider for the snackbar component.

###  Usage

```typescript jsx
import { ThemeProvider } from "@tracktor/design-system";

const App = () => (
  <ThemeProvider theme="dark">
    ...
  </ThemeProvider>
);

export default App;
```

### Props

| Name                    | Type                              | Required | default            | Description                      |
|-------------------------|-----------------------------------|----------|--------------------|----------------------------------|
| `fullHeight`            | boolean                           | false    | true               | Set html and body height to 100% |
| `includeCssBaseline`    | boolean                           | false    | true               | Include CssBaseline              |
| `includeScrollBarStyle` | boolean                           | false    | true               | Include scroll bar style         |
| `theme`                 | "dark" or "light" or ThemeOptions | false    | light              | Theme to use                     |
| `font`                  | object                            | false    | `{ import: true }` | Font options                     |
