# `useIsFeatureEnabled`

This hook return `true/false` if user hase given feature.

### Basic usage

```typescript jsx
import { useIsFeatureEnabled } from "design-system";

const hasCheckout = useIsFeatureEnabled({ features: ["order", "checkout"], name: "order" }); // return true
const hasBill = useIsFeatureEnabled({ features: ["order", "checkout"], name: "bill" }); // return false
```

### Provide user features

Put `<FeatureEnableProvider>` to provide user `features`<br/>
In this example above static data is provided, but you can fetch user features.

```typescript jsx
import React from 'react'
import Home from "./Home";
import { FeatureEnableProvider } from 'design-system';

const App = () => (
    <FeatureEnableProvider features={["order", "checkout"]}>
      <Home />
    </FeatureEnableProvider>
);

export default App;
```

Then anywhere in your application like `home.tsx` you can use the hook without give features

```typescript jsx
import { useIsFeatureEnabled } from "design-system";

const hasCheckout = useIsFeatureEnabled({ name: "checkout" }); // return true
const hasBill = useIsFeatureEnabled({ name: "bill" }); // return false
```
