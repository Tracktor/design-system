# `useColorMode`

This hook allows switch theme color mode

### Provide color mode
Put `<ColorModeProvider>` around your application and `<ColorModeConsumer>` to provide `colorMode` to `<ThemeProvider>`

```typescript jsx
import { ColorModeConsumer, ColorModeProvider, ThemeProvider } from "design-system";
import Home from "./Home";

const App = () => (
  <ColorModeProvider>
    <ColorModeConsumer>
      {({ colorMode }) => (
        <ThemeProvider theme={colorMode}>
          <Home />
        </ThemeProvider>
      )}
    </ColorModeConsumer>
  </ColorModeProvider>
);

export default App;
```

### Toggle color mode
Once `colorMode` is provided, you can toggle theme color mode anywhere in your application

```typescript jsx
import { Button, useColorMode } from "design-system";

const Home = () => {
  const { toggleColorMode } = useColorMode();

  return (
    <Button onClick={toggleColorMode}>
      Toggle theme color mode
    </Button>
  );
};

export default Home;
```
