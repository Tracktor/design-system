# Styles
The system prop `sx` that allows defining system overrides as well as additional CSS styles or
use`Emotion-based` styling solution provided by MUI to style your components.

### System Prop

```typescript jsx
import { Box } from "@mui/material";

<Box
  sx={{
    "&:hover": {
      backgroundColor: "blue",
    },
    backgroundColor: "red",
    height: 150,
    width: 150,
  }}
/>
```

### Emotion based

```typescript jsx
import { Box, css } from "@mui/material";

const myCss = css({
  ":hover": {
    backgroundColor: "blue",
  },
  backgroundColor: "red",
  height: "150px",
  width: "150px",
});

<Box css={myCss}/>
```
> __warning__ : If you want test your component inside story which includes emotion css,<br/>
put `/** @jsxImportSource @emotion/react */` at the top of your story file

### Global style
For injecting global style you can use `GlobalStyles` and `css` from Material UI. It is an alias of `Global` from emotion css.

```typescript jsx
import { GlobalStyles, css } from "@mui/material";

<GlobalStyles
  styles={{
    ".some-class": {
      color: "red",
      textAlign: "center",
    },
  }}
/>
```

Or `css` with `GlobalStyles`

```typescript jsx
import { GlobalStyles, css } from "@mui/material";

<GlobalStyles
  styles={css`
    .some-class {
      color: red;
    }
  `}
/>
```
