# Styles
Design System components use the system prop `sx` that allows defining system overrides as well as additional CSS styles or
use`Emotion-based` styling solution provided by MUI to style your components.

### System Prop

```typescript jsx
import { Box } from "design-system-tracktor";

const styles = {
  box : {
    "&:hover": {
      backgroundColor: "blue",
    },
    backgroundColor: "red",
    height: 150,
    width: 150,
  }
}

<Box sx={styles.box} />
```

### Global style
For injecting global style you can use `GlobalStyles` and `css` from Material UI. It is an alias of `Global` from emotion css.

```typescript jsx
import { GlobalStyles, css } from "design-system-tracktor";

<GlobalStyles
  styles={{
    ".some-class": {
      color: "red",
      textAlign: "center",
    },
  }}
/>
```

Or `css` with `GlobalStyles`

```typescript jsx
import { GlobalStyles, css } from "design-system-tracktor";

<GlobalStyles
  styles={css`
    .some-class {
      color: red;
    }
  `}
/>
```
